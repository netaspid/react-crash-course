## Блок 3. Композиция 

11. **Работа через children** - компонент-обёртка `Panel` с заголовком и телом: заголовок через prop, содержимое через `children`. Снаружи передаем в children произвольную JSX разметку.

12. **Кастомный хук `useLocalStorage`** - написать хук для работы с local storage. реализовать методы getStoredValue , setValue, removeValue, использовать useEffect для синхронизации между окнами/вкладками. на одной комбинации хоста-клиента local storage должен быть консистентным. Для примера работы реализации сделать функционал переключения темы сайта(с `dark` на `light`). В ui jsx не должно быть бизнес-логики работы с хранением значений - вся работа через хук.

```
import useLocalStorage from './useLocalStorage';

const ThemeSwither: React.FC = () => {
  const [theme, setTheme, removeTheme] = useLocalStorage<'light' | 'dark'>(
    'app-theme',
    'light'
  );

  const toggleTheme = () => {
    setTheme(theme === 'light' ? 'dark' : 'light');
  };

  return (
    <div style={{ 
      backgroundColor: theme === 'dark' ? '#333' : '#fff',
      color: theme === 'dark' ? '#fff' : '#333',
      padding: '20px',
      minHeight: '100vh'
    }}>
      <h1>Текущая тема: {theme}</h1>
      <button onClick={toggleTheme}>
        Переключиться в {theme === 'light' ? 'темную' : 'светлую'} тему
      </button>
      <button onClick={removeTheme} style={{ marginLeft: '10px' }}>
        Сбросить настройки тем
      </button>
    </div>
  );
};

export default ThemeToggle;
```
